Por que usamos o Z-score?

  • Para comparar dados de distribuições diferentes.
  • Para usar tabelas de probabilidade que só funcionam com média = 0 e desvio padrão = 1.
  • Para aplicar testes estatísticos como o Kolmogorov-Smirnov, que compara os dados com essa forma padrão da normal.

 

 

Testes de Normalidade

Os testes de normalidade aplicados têm como objetivo verificar se os dados de consumo mensal se distribuem de forma semelhante a uma distribuição normal.

  • Shapiro-Wilk: avalia se os dados se desviam significativamente da normalidade, sendo mais sensível para amostras pequenas e moderadas.
  • D’Agostino e Pearson: combina medidas de assimetria e curtose para testar a normalidade.
  • Kolmogorov-Smirnov (KS): compara a distribuição empírica dos dados com uma distribuição teórica normal padrão (Z-score).

Resultados dos testes:

  • Shapiro-Wilk: Estatística , p-valor
  • D'Agostino e Pearson: Estatística , p-valor
  • Kolmogorov-Smirnov (KS): Estatística , p-valor

 

Em todos os testes, os p-valores são superiores a , o que indica que não há evidência estatística para rejeitar a hipótese de normalidade. Portanto, os dados podem ser considerados normalmente distribuídos.

Relevância da Distribuição Normal

A Distribuição Normal tem grande importância em estatística, sendo a base do Teorema Central do Limite, que afirma que, sob certas condições, a soma de variáveis aleatórias independentes e identicamente distribuídas tende a uma distribuição normal, independentemente da distribuição original dos dados. Por isso, quando os dados observados de um fenômeno natural ou de sistemas artificiais podem ser interpretados segundo essa distribuição, tem-se facilidades para utilizar esses dados para fins práticos, com elevada precisão nas estimativas dos valores esperados para o objeto de análise.

🧠 O que esse gráfico revela?

  • Ambas as curvas começam próximas de zero (valores muito baixos de consumo são raros) e se aproximam de 1 (valores altos de consumo estão no fim da distribuição).
  • Pequenas diferenças entre as curvas indicam desvios entre o comportamento real dos dados e o modelo normal.
  • Por exemplo:
    • Se a curva azul estiver acima da vermelha em um ponto, significa que mais dados reais estão abaixo daquele valor do que o esperado pela normal.
    • Se estiver abaixo, significa que a normal "superestima" a quantidade acumulada de valores até ali.

Conclusão e Aplicação ao Projeto

A visualização gráfica e os testes estatísticos indicam que os dados de consumo mensal se ajustam adequadamente a uma distribuição normal com nível de significância de 5%. Com isso, é possível aplicar modelos estatísticos baseados nessa distribuição para projeções de consumo.

A partir disso, é possível definir com segurança vazões médias e máximas, volumes de reservatórios, margens de segurança operacionais, entre outros elementos fundamentais do projeto.
